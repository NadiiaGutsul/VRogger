# VRogger
VRogger is based on the classic “Frogger,” but reimagined as a real-time cooperative game for one VR player and one PC player. One player controls the frog on the PC using familiar arcade-style controls, but some roads and rivers cannot be crossed alone. 

That’s where the other player comes in—using VR, they must help the frog navigate as many levels as possible by utilizing water lilies and traffic cones. Through teamwork, skill, and good timing, VRogger transforms a simple arcade concept into a cooperative challenge where one thing matters: You must save the Frog!
